Procedural Planets

Dynamic and realtime procedural generation of a planet based on some generation parameters on the web using webgl 2.0. This app is hardware accelerated leveraging the GPU but will run smoothly on most modern hardware.

WebGL, Javascript, HTML/CSS

Super Resolution GAN

Intelligently upsizing images of any domain using state of the art ai technology. Although this model has been trained on a dataset of flowers, it can be finetuned on a domain of any images. This work is an extension of Nvidia's SRGAN specifically focusing on an improvement of the loss function.

Python, Pytorch

AI Denoise

In this project, I try several techniques to train a UNet to remove noise from images that have been generated using a raytracing pipeline with number of samples per pixel.

Python, Pytorch

Deep Dream

A pytorch implementation of google's deep dream algorithm focusing mainly on the math and correctness of the algorithm. The purpose of this project is to outline how the activations of the layers relate to the input image.

Python, Pytorch

Meadow

A desktop application to be used as a voxel editor to construct 3D works of art using arbitrarily colored blocks. The original intent behind this project was for it to be used alongside a generative adversarial network to experiment with 3 dimensional output.

C++, OpenGL, GLM, GLFW

The Towers

Open source multiplayer minecraft-like team-based game much like capture the flag. Incorporates many of the mechanics present in minecraft such as block placing/breaking, crafting, climbing etc but also adds more such as shooting and sliding.
